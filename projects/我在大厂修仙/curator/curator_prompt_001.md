## 第1章正文

```
# 第一章：觉醒

凌晨三点，办公室的日光灯还在亮着。

林默的手指在键盘上机械地敲击，眼睛盯着屏幕上密密麻麻的代码，瞳孔已经开始涣散。

"林默，进度怎么样了？"

张伟的声音从背后传来，带着一股让人作呕的优越感。

林默没回头，只是哑着嗓子说："还……还在debug……"

"还在debug？"张伟走到他工位旁边，俯视着他，"这个项目下周就要上线，你现在还在debug？你知道这个项目值多少钱吗？"

林默的手指停在键盘上，他能感觉到自己的心跳在加速，太阳穴在突突地跳。

"我……我已经连续加班三天了……"

"三天？"张伟笑了，"你知道我为了这个项目熬了多少夜吗？你知道这个项目对公司意味着什么吗？你知道这个项目能给公司带来多少营收吗？"

林默没说话，他只是低头看着屏幕，眼前开始发黑。

突然，他的脑海里闪过一个画面。

那是一个金碧辉煌的宫殿，他坐在龙椅上，俯视着跪在地上的群臣。

"吾乃仙帝……"一个声音在他脑海里回响。

林默愣住了，他揉了揉眼睛，以为自己出现了幻觉。

但画面越来越清晰，他看到了自己的前世——仙界统治者，万仙之首。

"这……这是怎么回事？"

他感觉到自己的丹田里有一股暖流在涌动，沿着经脉流向四肢百骸。

"灵气……"他喃喃自语，"这是灵气……"

他睁开眼睛，看向张伟，发现自己的视野变得不同了。

他能感知到张伟的每一个动作，每一个表情，甚至每一个心跳。

"神识……"他喃喃自语，"这是神识……"

张伟还在喋喋不休地说着，但林默已经听不进了。

他感觉到自己的修为在恢复，封印在松动。

"仙帝转世……"他喃喃自语，"原来我是仙帝转世……"

他抬起头，看向张伟，嘴角勾起一抹冷笑。

"张伟，"他说，"你以为你赢了？不，游戏才刚刚开始。"

张伟愣住了："什么？"

"没什么，"林默摇摇头，"我只是在想，这个项目，我能完成。"

张伟的嘴巴张了张，想说什么，但又说不出来。

"行……行吧，"他挥挥手，"继续干，明天早上我要看到成果。"

林默点点头，目送张伟离开。

然后，他闭上眼睛，开始用神识感知周围的一切。

他能感知到整栋大楼里的每一个WiFi信号，每一个电子设备，甚至每一个生命体。

"有意思，"他喃喃自语，"真有意思。"

他睁开眼睛，看向屏幕，发现自己的代码已经完成了。

"原来……这就是神识的力量。"

他站起来，走到窗边，看向窗外的城市。

夜色如墨，星光点点。

"仙界……"他喃喃自语，"我一定会回去的。"

突然，他感觉到一股强大的灵气波动从顶楼传来。

"那里……"他喃喃自语，"有什么东西在等我……"

```

## 规则

1. 只报告本章实际发生的变更，不要推测未来
2. machine_verifiable=true 的变更才写入硬约束（位置、生死、数值变化）
3. 角色的情感变化、态度变化标记为 machine_verifiable=false（软状态）
4. 伏笔状态：open→planted（埋下）→payoff（收束）→abandoned（放弃）
5. 信息差更新：谁在本章知道了什么新信息
