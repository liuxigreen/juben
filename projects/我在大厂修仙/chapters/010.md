# 第十章：当众打脸

下午两点，技术评审会。

会议室里坐满了人——张伟、李薇、王浩，还有几个前端和后端程序员。陈远山没有来，但林默的神识感知到他的灵气在顶楼延伸下来，在监听这个会议。

张伟站在白板前，手里拿着一份代码审查报告。

"我昨天review了林默的代码，"他说，"发现了很多问题。"

他翻开报告，开始念：

"第一，代码注释太少，不利于团队协作。第二，变量命名不规范，没有使用驼峰命名法。第三，代码结构不清晰，需要重新设计。"

他念完，抬起头，看着林默。

"林默，你有什么要说的吗？"

林默站起来，走到白板旁边。

"张伟，"他说，"你说的这三个问题，都是错的。"

会议室里安静了。

"第一，代码注释。"林默说，"我的代码不需要注释，因为代码本身就是最好的文档。你看不懂，是因为你的技术能力不够。"

张伟的脸色变了。

"第二，变量命名。"林默继续说，"驼峰命名法是上世纪的产物。现在业界主流是蛇形命名法，我的代码用的就是蛇形命名法。你连这个都不知道，怎么做技术主管？"

张伟的脸开始发红。

"第三，代码结构。"林默说，"我的代码结构是微服务架构，每个服务都是独立的，可以单独部署和扩展。你说'代码结构不清晰'，是因为你还在用单体架构的思维来看微服务。"

他走到白板前，拿起笔，在白板上画了一个架构图。

"这是公司的技术架构，"他说，"三年前的设计，现在还在用。你知道这个架构有什么问题吗？"

张伟没有回答。

"这个架构有四十七个高危漏洞，"林默说，"每个漏洞都可以让黑客入侵公司的核心数据库。你知道这些漏洞吗？"

张伟的脸色变得苍白。

"你……你怎么知道的？"他的声音有点发抖。

"我看了整个系统的代码。"林默说，"不只是这个项目的，是公司所有项目的。"

他放下笔，转身看着张伟。

"张伟，"他说，"你不是一个合格的技术主管。你的技术能力不行，管理能力也不行。你唯一会做的，就是用PUA的方式让下属加班，用画大饼的方式让下属卖命。"

会议室里鸦雀无声。

张伟的嘴巴张了张，想说什么，但最终只是转身离开了会议室。

"叮。"

半透明的界面出现在林默眼前：

```
【任务完成】
任务名称：怼领导一次
结果：成功
奖励：元婴期突破资格

是否立即突破？
```

林默点击了"是"。

瞬间，一股强大的灵气在他体内爆发。他的丹田开始膨胀，经脉开始扩张，一颗紫色的丹田在他体内形成——元婴期。

但突破的代价也来了。

他的鼻子开始流血，眼角开始渗血，耳朵里也开始嗡嗡响。他的身体承受不住这么快的突破速度，多余的灵气从他的毛细血管中渗出，带着血丝流了下来。

他深吸一口气，稳住了自己的灵气。

"元婴期。"他低声说，"终于到了。"

他走出会议室，沿着走廊往工位走。同事们用一种奇怪的眼神看着他——有敬畏，有恐惧，还有一种他读不懂的东西。

他回到工位坐下，打开电脑。

屏幕上出现了一条新的消息：

```
【仙帝系统 v1.0】
恭喜你，仙帝。

你已经完成了所有觉醒任务。
你的境界：元婴期。
你的能力：WiFi炼丹、灵气感知、神识操控。

下一步：前往仙门，找回你的记忆。

警告：仙门不会欢迎你。你需要做好战斗准备。
```

林默看着这条消息，嘴角动了动。

"仙门，"他说，"我来了。"

他站起来，走到窗边。外面的城市在夕阳下变成了金色，万家灯火像一片星海。

他的神识穿透玻璃，穿透云层，穿透大气层，直接感知到了远处那座山峰上的灵气波动——仙门的所在地。

就在这时，他的神识捕捉到了一个异常信号。

那是一个微弱的、断断续续的灵气波动，来自公司地下三层——一个他从未去过的地方。那个波动的频率很特殊，不是仙门的，也不是魔界的，而是一种他从未见过的、古老而强大的力量。

"那是什么？"他低声说。

他集中神识，试图穿透地下三层的墙壁。但他发现，那里有一层强大的结界，连他的神识都无法穿透。

"有意思。"他说，"公司地下，还藏着什么秘密？"

他的神识继续探测，终于在结界的边缘捕捉到了一个微弱的信号——那是一个声音，一个女人的声音，断断续续，像是在呼救：

"救……救我……仙帝……救我……"

林默的瞳孔收缩了一下。

"那个声音……"他低声说，"是谁？"

他感觉到自己的心跳在加速。那个声音让他有一种奇怪的感觉——一种熟悉的感觉，一种让他心痛的感觉。

"我前世……"他喃喃道，"认识这个人吗？"
