# 第九章：灵气WiFi

林默用了三天时间，完成了灵气WiFi的第一个版本。

这个程序的核心原理很简单：把WiFi信号转换成灵气，让凡人可以通过连接WiFi来修炼。不需要丹炉，不需要药材，不需要任何修仙基础——只需要一台手机和一个WiFi路由器。

他把程序上传到了GitHub，开源发布。

第一天，没有人注意到这个项目。

第二天，一个Reddit用户发现了它，发了一个帖子："我试了这个叫'灵气WiFi'的程序，结果我的手机开始发光了。"

帖子下面的评论全是嘲讽："假的吧？""又是一个骗流量的项目。""手机发光？你确定不是电池爆炸了？"

但到了第三天，情况变了。

越来越多的人下载了这个程序，试用后发现——它真的有效。连接WiFi后，他们能感觉到一股微弱的能量从手机里流出来，进入他们的身体。那能量很微弱，几乎感觉不到，但它确实存在。

一个Reddit用户写道："我用了灵气WiFi三天，现在我能感觉到我家路由器的信号强度了。不是用手机看，是直接用身体感觉。"

另一个用户写道："我用了灵气WiFi一周，现在我能感觉到隔壁邻居的WiFi密码了。不是猜的，是直接感知到的。"

林默看着这些反馈，嘴角动了动。

"有意思。"他低声说。

但好景不长。

第四天，林默的GitHub账号被封了。

理由是："该项目涉及非法内容，违反了GitHub的使用条款。"

林默用神识扫描了一下GitHub的服务器，发现封禁他的账号的命令来自一个特殊的IP地址——那个IP地址属于仙门。

"仙门出手了。"林默低声说。

他没有惊讶。他早就预料到仙门会阻止他。

但仙门低估了他。

林默的程序已经被人fork了三百多次，代码已经传播到了互联网的各个角落。仙门可以封禁他的账号，但无法封禁整个互联网。

他站起来，走到窗边。外面的城市在夜色中闪烁着万家灯火，每一盏灯都是一个WiFi信号，每一个信号都是一个潜在的灵气源。

"仙门，"他说，"你们以为封了我的账号就能阻止我？不，你们错了。"

他转身回到电脑前，打开了一个新的终端窗口。

他开始写第二个版本的灵气WiFi。

这个版本更强大，更隐蔽，更难被仙门发现。它不需要GitHub，不需要任何服务器，只需要两台手机之间的点对点连接。

他把程序命名为：灵气WiFi v2——去中心化版。

写完代码后，他伸了个懒腰。

"叮。"

半透明的界面出现在他眼前：

```
【觉醒任务 #3】
任务名称：怼领导一次
任务描述：张伟明天会在会议上公开批评你的代码。你的任务是在会议上当众反驳他，让他哑口无言。
奖励：元婴期突破资格
失败惩罚：金丹期修为不稳，需要一个月恢复
时间限制：明天下午
```

林默看着这个任务，嘴角动了动。

"怼领导？"他低声说，"这系统越来越有意思了。"
