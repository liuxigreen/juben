# 第七章：觉醒任务

第二天早上，林默打开电脑，发现桌面上多了一个图标。

图标是一个金色的火焰，下面写着"仙帝系统"。

林默的神识扫描了一下这个图标——它不是普通的程序，而是一个灵气程序，需要用神识才能运行。

他用神识点击了图标。

一个半透明的界面出现在他眼前，只有他能看到：

```
【仙帝系统 v1.0】
欢迎回来，仙帝。

当前境界：筑基巅峰
下一境界：金丹期
突破条件：完成觉醒任务

【觉醒任务 #1】
任务名称：让产品经理闭嘴1小时
任务描述：产品经理李薇平均每小时打断程序员3次。你的任务是让她安静1小时。
奖励：金丹期突破资格
失败惩罚：修为倒退一个小境界
时间限制：今天
```

林默看着这个任务，眉毛挑了一下。

"让产品经理闭嘴1小时？"他对着屏幕喃喃，"这系统还挺有意思。"

他站起来，走向产品经理李薇的工位。

李薇正在和一个前端程序员讨论需求。她的声音很大，语速很快，每隔几秒钟就打断对方一次。

"但是这个交互逻辑不对……"

"你听我说完……"

"我觉得应该这样做……"

林默走到她旁边，用神识扫了一下她的情绪——焦虑、烦躁，还有一种想要证明自己的欲望。

"李薇，"林默说，"我有个技术问题想请教你。"

李薇转过头，看着林默。

"什么问题？"

"关于用户行为分析的。"林默说，"我想知道，用户在什么情况下会放弃使用一个产品？"

李薇的眼睛亮了。这是她最擅长的话题。

"这个问题问得好。"她说，"用户放弃一个产品，通常有三个原因……"

她开始滔滔不绝地讲起来。林默一边听，一边用神识操控周围的WiFi信号。

他悄悄地在李薇的电脑上运行了一个小程序——这个程序会模拟李薇的打字习惯，在她不注意的时候，自动回复同事的消息。

现在，李薇以为自己在和林默讨论技术问题，但实际上，她的同事们都以为她在正常工作。没有人会来找她，没有人会打断她。

一个小时后，李薇终于讲完了。

"……所以，用户放弃一个产品，根本原因是期望值和实际体验的差距。"她总结道。

"明白了。"林默说，"谢谢。"

他转身离开，回到工位。

半透明的界面再次出现：

```
【任务完成】
任务名称：让产品经理闭嘴1小时
结果：成功
奖励：金丹期突破资格

是否立即突破？
```

林默点击了"是"。

瞬间，一股强大的灵气在他体内爆发。他的丹田开始膨胀，经脉开始扩张，一颗金色的丹田在他体内形成——金丹期。

但突破的代价也来了。

他的眼角开始渗出血丝。不是普通的血丝，而是灵气溢出——他的身体承受不住这么快的突破速度，多余的灵气从他的毛细血管中渗出。

他用手擦了擦眼角的血，深吸一口气。

"金丹期。"他盯着掌心的血丝，"终于到了。"

他站起来，走到窗边。他的神识穿透玻璃，穿透云层，直接感知到了远处那座山峰上的灵气波动——仙门的所在地。

"仙门，"他说，"我来了。"

但就在这一秒，他的电脑屏幕上又弹出了一个新的任务：

```
【觉醒任务 #2】
任务名称：准时下班一次
任务描述：你已经连续加班7天了。你的任务是今天下午6点准时离开公司。
奖励：金丹期修为巩固
失败惩罚：灵气紊乱，需要3天恢复
时间限制：今天下午6点
```

林默看着这个任务，嘴角扯了一下。

"准时下班？"他敲了敲桌面，"这系统还挺懂我。"
