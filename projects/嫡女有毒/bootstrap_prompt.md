# Juben 角色与世界观填充任务

你是一个专业的剧本架构师。根据以下信息，为这个故事项目生成完整的角色卡和世界观设定。

## 故事前提
沈清辞是丞相府嫡女，前世被继母和庶妹联手陷害，死于非命。重生回三年前，她决定不再做任人摆布的棋子。这一世，她要夺回属于自己的一切，让害她的人付出代价。

## 意外变量（核心金手指）
重生回三年前，带着前世记忆

## 已有规则约束

### 因果约束（不可违反）
- 古代没有手机、互联网、外卖，不能出现任何现代通讯工具
- 女性在古代社会地位受限，女主的每一步突破都必须有合理的时代逻辑支撑
- 银两购买力要符合时代，不能'一掷千金'却毫无后果
- 古代信息传递靠驿站/信鸽，消息有延迟，不能秒传
- 朝堂规则：奏折、廷议、圣旨都有固定流程，不能随意'面圣'
- 医药水平有限，重伤不能三章痊愈
- 古代出行靠马/轿/步行，距离感要真实

### 反套路黑名单（禁止出现的情节）
- 女主穿越后随口背唐诗宋词震惊全场——古人也会写诗，没那么稀罕
- 女主用现代医学知识秒变神医——古代没有消毒液和抗生素
- 女主随口说出'民主''平等'引发朝堂震动——古人听不懂也不会当回事
- 男主是王爷/太子却天天闲着陪女主逛街——皇族有规矩有差事
- 丫鬟/侍女对主子直呼其名——古代主仆尊卑分明
- 女主在古代开奶茶店/火锅店发家致富——原材料和供应链从哪来？
- 圣旨/懿旨像微信消息一样随便发——拟旨有流程，用印有规矩
- 女主在宴席上当众跳舞/弹琴惊艳四座——古代大家闺秀抛头露面是失礼
- 穿越者用'现代思维'碾压所有人——古人的智慧不比现代人差
- 女主轻松进出皇宫如入无人之境——宫禁制度极其严格
- 女主被陷害时只会哭和等男主来救——有手有脚有脑子，自己想办法
- 男主在千钧一发之际从天而降——不是每一场危机都需要男主出场
- 女主明明有证据却不拿出来，非要等误会加深——这叫注水不叫剧情
- 女配疯狂嫉妒女主只为抢男人——女性之间的关系不止'争宠'这一种
- 女主摔倒必被男主接住——物理学不存在了？
- 女主生气就跑，男主追上来壁咚——2026年了，能不能有点新意？
- 女主真实身份是公主/郡主/神女——身份反转靠血统不靠能力是最大的懒惰
- 女主被退婚后对方后悔跪舔——退婚不是爽点，能力碾压才是
- 女主掏出信物/令牌让全场震惊——信物不等于实力
- 女主隐藏身份当丫鬟/乞丐，一朝翻身众人跪——身份落差要靠过程积累
- 男主冷面霸道只为女主一人融化——冷面人设需要有合理成因
- 女主和男主初次见面就心跳加速——铺垫不足的一见钟情是偷懒
- 误会推动感情：不听解释转身就走——成年人应该会沟通
- 男主后宫争宠模式——女频核心是女主成长，不是男主选妃
- 反派当众揭发女主，女主三句话反杀——权谋不是吵架比赛
- 女主在朝堂上舌战群儒——古代女性上朝堂需要极特殊的设定
- 反派的阴谋永远只差一步就被识破——真正的威胁是读者都看不穿的局

### 核心原则
- 女本位叙事：女主有独立人格和目标，不是男主的附属品
- 双强设定：男女主势均力敌，不是谁救谁
- 成长弧光：女主从'小我'到'家国'的价值递进
- 能力立人：先用能力证明自己，再谈感情

## 算法卡点（每章必须遵循的节奏）
- [3s_Hook] 0-100字: 动词+特写开局。禁止背景铺垫。必须在前100字内出现一个具体的感官冲击（坠落感/疼痛/重生的眩晕）
- [15s_Retention] 300-500字: 爆出核心信息差——主角知道但其他人不知道的关键事实。这是留住读者的炸弹
- [Mini_Tension] 600-800字: 一次小规模紧张对峙或试探，主角不能暴露自己知道太多
- [Emotion_Dip] 1000-1200字: 情绪下压——回忆前世的痛苦/看到还活着的已故亲人/意识到代价
- [50s_Cliffhanger] 1700-2000字: 断崖。必须在最后一句植入一个具体的未回答问题

## 设计要求

1. **角色必须有缺陷**：完美的主角没有故事。给主角一个真实的、影响剧情的弱点。
2. **反派必须有逻辑**：反派的动机不能是"天生坏"。给反派一个读者能理解（但不认同）的理由。
3. **关系必须有张力**：角色之间的关系不是静态的。标注出潜在的变化方向。
4. **信息差必须具体**：不要写"主角知道未来"，要写"主角知道某人在第X章会做某事"。
5. **符合已有的因果约束和反套路黑名单**：生成的内容不能违反上面列出的任何规则。

只输出JSON，不要输出任何其他文字。
